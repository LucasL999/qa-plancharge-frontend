# Plan de charge (front)
Une application web pour gérer le plan de charge de l'équipe QA

## Fonctionnalités


## Stack techniques
- Front : React / MUI
- Back : Node / Express
- DB : PostgreSQL
- Auth : Keycloak


## Installation du front

### 1. Cloner le projet 
'''bash
git clone 'lien du projet git récupérable depuis github/gitlab'
cd 'nom du dossier qui vient d'être clonné'
'''

### 2. Installation des dépendances
'''bash
npm install
'''

### 3. Lancement du front
'''bash
npm start
'''
